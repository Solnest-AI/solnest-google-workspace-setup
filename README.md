# Google Workspace MCP Setup

Connect Gmail, Google Calendar, Drive, Docs, and Sheets to Claude so it can actually read and act on your work.

A free setup guide from **Solnest AI**.

## How to use it

1. Download the zip from the Solnest AI Skool community (or the green **Code → Download ZIP** button above) and unzip it.
2. Open **Claude Code** (or the Claude Desktop app).
3. Drag the whole folder into the chat and say: **"Set this up."**

Claude reads the guide and walks you through it. You don't need to write any code.
